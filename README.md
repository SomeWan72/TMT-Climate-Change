# TMT-Climate-Change

For this analysis I have use the libraries listed below:

- devtools
- base64enc
- twitteR
- tm
- ggplot2
- ggmap
- twitteR
- httr
- wordcloud
- SnowballC
- RColorBrewer
- stringr
- lubridate
- data.table
- plyr
- rtweet
- dplyr 
- webshot
- htmlwidgets
- wordcloud2

It also includes the result generated (including one wordcloud with the most used words).
